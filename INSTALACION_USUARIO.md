# Instrucciones de Instalación - GC Game Launcher

## Para Usuarios Finales

### Requisitos del Sistema

- **Sistema Operativo**: Windows 10 o superior (64 bits)
- **Espacio en disco**: Mínimo 200 MB libres
- **Memoria RAM**: Mínimo 2 GB
- **No se requiere**: Node.js, Python, ni ninguna otra dependencia técnica

### Pasos de Instalación

1. **Descargar el Instalador**
   - Obtén el archivo `GC Game Launcher Setup 1.0.0.exe` del desarrollador
   - Guárdalo en una ubicación fácil de encontrar (por ejemplo, el Escritorio o la carpeta Descargas)

2. **Ejecutar el Instalador**
   - Haz doble clic en `GC Game Launcher Setup 1.0.0.exe`
   - Si aparece una advertencia de Windows Defender SmartScreen:
     - Haz clic en "Más información"
     - Luego en "Ejecutar de todas formas"
     - (Esto es normal para aplicaciones no firmadas digitalmente)

3. **Seguir el Asistente de Instalación**
   - El instalador te guiará paso a paso
   - Puedes elegir la carpeta de instalación (por defecto: `C:\Users\[TuUsuario]\AppData\Local\Programs\GC Game Launcher\`)
   - Elige si quieres crear accesos directos en el escritorio y menú inicio
   - Haz clic en "Instalar"

4. **Finalizar la Instalación**
   - Espera a que termine la instalación (solo toma unos segundos)
   - La aplicación se abrirá automáticamente al finalizar
   - ¡Listo! Ya puedes usar GC Game Launcher

### Primera Ejecución

La primera vez que abras la aplicación:
- Configura tu API Key de SteamGridDB (opcional, pero recomendado para obtener portadas de juegos)
- La aplicación detectará automáticamente tus juegos de Steam
- Puedes agregar juegos personalizados desde el menú

### Ubicación de Archivos

- **Aplicación instalada**: `C:\Users\[TuUsuario]\AppData\Local\Programs\GC Game Launcher\`
- **Datos y configuraciones**: `C:\Users\[TuUsuario]\AppData\Roaming\gclauncher\`
- **Accesos directos**: Escritorio y Menú Inicio

### Desinstalación

1. Ve a "Configuración" de Windows
2. Selecciona "Aplicaciones"
3. Busca "GC Game Launcher"
4. Haz clic en "Desinstalar"

O desde el Menú Inicio:
- Busca "GC Game Launcher" → "Desinstalar GC Game Launcher"

### Solución de Problemas

**Problema**: Windows bloquea la instalación
- **Solución**: Haz clic en "Más información" → "Ejecutar de todas formas"

**Problema**: No se detectan juegos de Steam
- **Solución**: Asegúrate de tener Steam instalado y haber iniciado sesión al menos una vez

**Problema**: La aplicación no se abre
- **Solución**: 
  - Verifica que tu sistema cumpla con los requisitos
  - Intenta ejecutar como administrador
  - Reinstala la aplicación
